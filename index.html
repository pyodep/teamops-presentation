소규모 팀

목적 · PURPOSE

GitHub · Azure를 팀과 연결

따로 놀던 GitHub · Azure를 팀의 작업 흐름에 연결 — 회의에서 나온 일부터 개발·배포까지 하나로 이어줍니다.

시장경쟁력 · EDGE

빈틈을 차지하는 포지션

개인용 도구엔 없는 팀 협업, 대기업용 도구엔 없는 간결함·합리적 비용. 소규모 팀 시장의 빈틈을 정확히 공략.

Scenario · 03

AI 질의 — 회의록 기반 검색, 근거까지

지난 회의에서 AI 모델은 뭐 쓰기로 했어? RAG 파이프라인은 어떻게 설계했어? 마감이 바뀐 할 일이 있었어?

발표 자료 캡처는 디자인 담당이 맡기로 했어요. 관련 결정은 아래 회의록에 있어요. (회의록: 2026-07-08 발표전 최종점검)

근거 회의록 (2)
발표전 최종점검 2026-07-08"화면 캡처는 민지가 이번 주까지 정리하기로…"
발표 리허설 1차 2026-07-07"데모 시나리오·캡처 자료 준비 확인"

회의록(meetings/)을 근거로 답하는 RAG 검색 — 답변마다 근거 회의록을 함께 보여주고, 카드를 누르면 원문으로 이동합니다.
